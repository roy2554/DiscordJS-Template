# DiscordJS-Template
DiscordJS Bot Template
```console
$ npm install
```
Type `npm install` to install Library.

## How To Install
Type
```console
$ git clone https://github.com/roy2554/DiscordJS-Template
```

## How To Run
Type
```console
$ node index.js
```
or
```console
$ node .
```
in console
